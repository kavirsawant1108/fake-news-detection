# Fake News Detection System

A machine learning-based system to detect whether a news article is real or fake. This project uses Natural Language Processing (NLP) techniques to analyze news text and classify it accurately.

---

## Features

- Classifies news as **Real** or **Fake**
- Uses NLP techniques for text preprocessing
- Machine learning model for prediction
- User-friendly interface for testing news articles

---

## Tech Stack

- **Backend:** Python (Flask / Django / Streamlit)
- **Frontend:** HTML, CSS, JavaScript
- **Machine Learning:** Scikit-learn, NLTK
- **Database:** SQLite / MySQL (if applicable)
